CTFd 中文汉化版
---
目前拥有 CTFd 的中文版本如下：
- 2.5.0
- 3.0.0
- 3.0.1

同时还将CTFd引用的外部css放到了gitee上，解决了国内访问量突增时，引用的外部链接挂掉，平台加载速度缓慢的问题，以加速访问。


1.CTFd详细部署方式请看[官方Wiki](https://docs.ctfd.io/)

2.用法：

CTFd部署成功后将此项目中的themes和plugins替换到 CTFd项目的CTFd下<br>
即下面的这两个目录
```
./CTFd/CTFd/themes
./CTFd/CTFd/plugins
```
